MSI Product Announcement - February 22, 2026

MSI today announced the EliteBook X, featuring up to 64GB of DDR5 memory, targeting Gaming.

PRODUCT SPECIFICATIONS:
- Model: EliteBook X
- Memory Options: 16GB/32GB/64GB
- Processor: Apple M4
- Price: $2852
- Availability: Immediate

MARKET POSITIONING:
MSI positions the EliteBook X as a premium Gaming solution, emphasizing memory capacity as a key differentiator.

Soldered memory limits upgradeability
